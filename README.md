## About Temporis Arts

Temporis Arts는 원판 위를 회전하는 판정선을 따라, 그리고 음악에 맞춰 키를 누르는 게임입니다. 지정된 키 중 어느 키를 눌러도 됩니다 - 다만 갯수만 맞다면요!  
특이한 방식의 판정으로 인해, 일반적인 리듬게임을 즐기던 사람이어도 충분히 새로운 경험을 할 수 있습니다. 40곡 이상, 80개 이상의 패턴이 준비되어 있습니다!  
<br>
처음에 잠겨 있는 곡이나 패턴은 "메모리얼"을 통해 해금할 수 있습니다.  
밤하늘에 펼쳐진 별들을 발견하고, 새로운 곡을 획득해보세요!
